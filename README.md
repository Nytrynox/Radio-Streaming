# 📻 Radio - Next.js Application

A modern web radio streaming application built with Next.js and TypeScript.

---

## 🌟 Overview

A feature-rich radio streaming platform with a modern UI, real-time streaming, and multi-library support.

## ✨ Features

- **Live Streaming** - Real-time radio broadcasts
- **Modern UI** - Next.js with TypeScript
- **Multi-Library** - Extensive music library
- **Responsive** - Works on all devices

## 📁 Project Structure

```
radio/
├── app/             # Next.js app router
├── components/      # UI components
├── lib/             # Utilities
└── package.json     # Dependencies
```

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Access at: `http://localhost:3000`

## 🔬 Technology Stack

| Component | Technology |
|-----------|------------|
| Framework | Next.js 14 |
| Language | TypeScript |
| Styling | CSS/Tailwind |
